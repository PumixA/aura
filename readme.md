**Aura (Mirroir connecté)**

Ceci est un projet Django tournant sur une bdd posgres ayant pour objectif de fontionner sur un rasberry pi 3b+. <br>
L'objectif de ce projet est de concevoir un mirroir tactile connecté. <br>
La date de début de ce projet est située a septembre 2024. <br>
Ce readme sera complété au fur et a mesure pour expliquer les étapes d'installation et de configuration. <br>
Prévu pour fonctionner sur linux, spécifiquement un rasberry avec un ecran aux dimensions 1080x1920. <br>
